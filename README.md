# GDIZ Smart Service Node

Produit de service connecté pour restauration, hôtellerie et accueil premium.

## Finalité
Transformer chaque table ou point de service en canal de :
- service opérationnel
- expérience mobile client
- capture prospect
- relance CRM / WhatsApp
- pilotage manager

## Proposition de valeur
Le produit ne vend pas un simple bouton.
Il transforme un point physique en interface de service et d’acquisition client.

## Cœur du produit
- 4 actions : Appeler, Service/Boisson, Addition, Annuler
- QR unique par table ou point de service
- expérience média sur le téléphone du client
- capture de lead avec consentement
- dashboard manager pour service + conversion

## Architecture cible
### 1. Node physique
- boîtier stable
- 4 boutons
- LED d’état
- USB-C charge et maintenance
- identifiant device unique

### 2. Expérience mobile
- landing légère
- menu / offre / contenu d’attente
- CTA WhatsApp
- formulaire prospect
- tracking table + site

### 3. Exploitation
- réception staff
- historique des demandes
- temps moyen de réponse
- taux scan → lead
- taux clic WhatsApp

## Démo cible
1. Le client appuie sur Appeler
2. Le staff reçoit la demande
3. Le client scanne le QR
4. Il ouvre la landing liée à sa table
5. Il laisse prénom + WhatsApp
6. Le manager voit demande, scan, lead et conversion

## Positionnement
- pas d’audio local dans le boîtier en V1
- tout le média passe sur le téléphone du client
- produit vendu comme système d’expérience et de revenu
